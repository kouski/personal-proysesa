# Objetivo del proyecto

Crear una aplicación de gestión de Personal para la empresa Proysesa, usando NextJS (App Router) y Tailwind.

El proyecto ya está creado y con tailwind instalado, pero sin estilos ni funcionalidades. El objetivo es implementar el diseño y las funcionalidades básicas de una app de gestión de personal moderna, premium y minimalista, siguiendo exactamente los diseños presentados en las referencias que serán proporcionadas que se encuentran en la carpeta de "Antigravity" dentro del proyecto.
